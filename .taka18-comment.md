### O que falta — bloqueio em ação humana, não em código

Os passos abaixo exigem login no painel Vercel + acesso à conta GitHub do dono do repo. Nenhum agente desta company tem essas credenciais. Ação pelo **Deyvysson** (5 min):

1. Acessar [vercel.com/new](https://vercel.com/new) → "Import Git Repository" → autorizar GitHub → escolher `deyvyssonbr/pizzaria-premium`.
2. Framework Preset: **Other** · Root Directory: `./` · Build Command: vazio · Output Directory: vazio (default). O `vercel.json` já está commitado.
3. Em **Environment Variables**, adicionar:
   - Nome: `MP_ACCESS_TOKEN`
   - Valor: token de produção do Mercado Pago (não comitar em lugar nenhum)
   - Environment: **Production** (e Preview se quiser testar PRs).
4. Click **Deploy**. Em ~1 min sai uma URL tipo `https://pizzaria-premium-xxxxx.vercel.app`.
5. Em **Settings → Domains**, fixar o alias amigável `pizzaria-premium.vercel.app` (ou outro nome disponível).
6. Testar PWA no Chrome desktop e mobile (instalar como app), abrir DevTools → Console e confirmar zero warnings de mixed content.
7. Colar a URL final + screenshot da instalação PWA neste thread.
